# Rag-система Нейро-сотрудника
В рамках проекта была спроектирована и реализована RAG-система в формате «нейро-сотрудника», ориентированная на анализ требований, рисков и регламентов на основе ограниченной базы знаний.

В ходе работы:

    Установлена роль ИИ-ассистента с фиксацией строгих правил его работы;
    сформирована и структурирована база знаний, разделённая по типам документов;
    выбран и реализован подход с использованием векторного поиска (FAISS) и локальной русскоязычной LLM;
    проведена поэтапная трассировка ответов, зафиксировано исходное (baseline) поведение;
    Основными недостатками стали: частые ложные срабатывания фильтра безопасности, риск "галлюцинаций" в ответах на сравнительные запросы и избыточная самопроверка;
    Реализованы архитектурные улучшения, включая двухэтапную генерацию ответов (основной ответ + проверка), явное формирование контекста, нормализацию отказов и упрощенный переформулировку запросов.
